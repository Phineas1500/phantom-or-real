## Likely Figure Narrative

1. Start with behavior: deeper ontology reasoning is where the model fails.
2. Show raw pre-generation activations predict that failure beyond metadata.
3. Show sparse dictionaries partially expose the signal but do not match raw.
4. Use reconstruction/error to make the sparse-dictionary caution sharp.
5. Use causal figures to show predictive directions are not reliable steering
   handles and that full-state patching has an asymmetric structure.

## Design Notes

- Main AUC figures use a restrained palette: gray metadata, blue raw, green
  residual SAE, purple sparse concat, red reconstruction error, orange
  transcoders.
- Causal figures intentionally plot counts and margins rather than only prose
  summaries, because the key story is a calibrated null/asymmetry.
- Avoid putting every sparse ablation in the main text. Use the sparse
  progression and exact-hook site figures as appendix support for the compact
  raw-vs-sparse overview.
